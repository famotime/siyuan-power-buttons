# 配置界面预览图标尺寸调整说明

- 文档日期：2026-04-17
- 适用范围：插件设置页“位置预览”中的按钮图标显示尺寸

这份文档只回答一个问题：配置界面里，自定义按钮和原生按钮在预览区的图标大小要去哪里改。

## 1. 先分清两层尺寸

设置页预览按钮的大小，实际上分成两层：

- 图标外框尺寸
- 图标本体尺寸

对应文件都在 `src/index.scss`。

## 2. 图标外框尺寸

位置：

- `src/index.scss`
- 选择器：`.workspace-chip__icon`

当前实现：

```scss
.workspace-chip__icon {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  width: 20px;
  height: 20px;
  border-radius: 7px;
  background: color-mix(in srgb, var(--b3-theme-primary) 12%, transparent);
}
```

这里控制的是预览区里图标的背景小方块大小，不直接决定 SVG 画出来有多大。

## 3. 图标本体尺寸

位置：

- `src/index.scss`
- 选择器：`.workspace-chip__icon .siyuan-power-buttons__icon, .workspace-chip__icon svg`

当前实现：

```scss
.workspace-chip__icon .siyuan-power-buttons__icon,
.workspace-chip__icon svg {
  width: 18px;
  height: 18px;
}
```

这里才是配置界面预览区里图标真正显示多大的关键位置。

之所以不能只改 `.siyuan-power-buttons__icon`，是因为：

- 一部分自定义图标会带 `.siyuan-power-buttons__icon`
- 另一部分图标，尤其是 IconPark 原始 SVG，只会直接输出 `<svg width="1em" height="1em">...`
- 所以预览区必须直接约束 `.workspace-chip__icon svg`，否则不同来源的图标会显示成不同大小

## 4. 推荐修改方法

### 方案 A：只放大图标，不改外框

适合想让图标更清晰，但不想让预览按钮显得更占地方的情况。

```scss
.workspace-chip__icon .siyuan-power-buttons__icon,
.workspace-chip__icon svg {
  width: 20px;
  height: 20px;
}
```

注意：

- 这种改法下，图标会更贴近外框边缘
- 如果图标显得太挤，再一起调整外框尺寸

### 方案 B：图标和外框一起放大

这是更稳妥的改法。

```scss
.workspace-chip__icon {
  width: 22px;
  height: 22px;
}

.workspace-chip__icon .siyuan-power-buttons__icon,
.workspace-chip__icon svg {
  width: 20px;
  height: 20px;
}
```

这个组合通常会比默认值更协调。

## 5. 常见误区

### 误区 1：只改 `.siyuan-power-buttons__icon`

不够。

原因：

- 不是所有预览图标都会带这个 class
- IconPark 原始 SVG 可能完全绕过这个 class

### 误区 2：只改 `.workspace-chip`

`.workspace-chip` 主要控制按钮容器、边距和字体，不是控制预览 SVG 实际宽高的关键点。

### 误区 3：把运行时按钮样式和设置页预览样式混在一起

`src/index.scss` 里还有这类选择器：

```scss
.siyuan-power-buttons__button
.siyuan-power-buttons__icon
```

它们更偏向插件运行时实际挂载到顶栏、状态栏、编辑区的按钮，不是设置页预览区的核心控制点。

## 6. 建议调整顺序

推荐按这个顺序改：

1. 先改 `.workspace-chip__icon .siyuan-power-buttons__icon, .workspace-chip__icon svg`
2. 如果图标显得拥挤，再改 `.workspace-chip__icon`
3. 改完后回到设置页，重点看顶栏、编辑区、状态栏这三个区域的观感是否一致

## 7. 当前默认值

当前仓库默认值是：

- 图标外框：`20px`
- 图标本体：`18px`

如果需要更醒目一些，可以先试：

- 图标外框：`22px`
- 图标本体：`20px`
