# 预置图标存储说明

## 概述

本插件的预置图标数据分布在两个核心文件中，采用「自动生成 + 手动补充」的分层架构。

## 文件结构

### 1. 自动生成目录 — `src/generated/iconpark-catalog.ts`

- **生成工具**：`tools/generate-iconpark-catalog.ts`
- **内容**：完整的 IconPark 图标元数据，每条记录包含 `value`、`name`、`label`、`category`、`keywords` 字段及 SVG markup
- **数据来源**：IconPark 图标库，按分类（如「通用」等）组织
- **维护方式**：通过运行生成工具自动更新，不建议手动编辑

### 2. 统一入口 — `src/shared/icon-catalog.ts`

从生成文件导入 IconPark 数据，并补充以下内容：

- **`COMMON_EMOJI_OPTIONS`**：常用 Emoji 列表（⚡🔍⚙️📝📌⭐📚🧩📎）
- **`LEGACY_BUILTIN_TO_ICONPARK`**：旧版内置图标标识到 IconPark 名称的映射表，用于向下兼容

## 相关文件
